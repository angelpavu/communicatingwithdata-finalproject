# NYC Airbnb Price Explorer

## Overview
This project is an interactive Shiny dashboard that analyzes Airbnb listings in New York City to answer one main question:

**What are you truly paying for?**

We used 48,895 listings across all five boroughs to understand what drives price differences.

## Key Findings
- **Neighborhood matters more than borough**  
  Prices vary heavily within boroughs, not just across them.

- **Room type is the biggest driver of price**  
  Entire homes, private rooms, and shared rooms are completely different price tiers.  
  You are paying for privacy and space.

- **Reviews do not increase price**  
  Listings with more reviews are slightly cheaper on avg.

## Features
- Data story section (NYT-style narrative)
- Interactive charts with filters (borough, price, room type)
- Clear visuals with labels and context
- User-driven exploration of pricing patterns
